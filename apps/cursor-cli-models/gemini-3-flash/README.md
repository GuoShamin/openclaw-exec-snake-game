# 自动驾驶贪吃蛇 (Snake AI)

这是一个基于 Web 技术实现的贪吃蛇小游戏，具备 AI 自动驾驶功能。

## 功能特点
- **自动驾驶**：使用 BFS 路径规划算法，结合安全余量检查（确保蛇头始终能找到通往蛇尾的路径），实现高效且持久的自动游玩。
- **移动端适配**：响应式画布设计，禁用页面滚动，适配各种屏幕尺寸。
- **GitHub Pages 友好**：纯静态资源（HTML/JS/CSS），无任何外部依赖，路径均为相对路径。

## AI 策略说明
1. **最短路径寻路**：使用 BFS 算法寻找蛇头到食物的最短路径。
2. **安全模拟**：在决定走最短路径前，先模拟移动一步，检查移动后蛇头是否仍有路径通往蛇尾。这能防止蛇进入无法逃脱的死胡同。
3. **跟随蛇尾**：若无法安全到达食物，则尝试移动向蛇尾。因为蛇尾移动后会腾出空间，这通常是最安全的避障策略。
4. **随机避障**：若以上策略均失效，则在周围寻找可行的空位。

## 本地运行方式
1. 将本项目所有文件下载到本地。
2. 使用任何静态服务器运行。例如：
   - 使用 VS Code 的 Live Server 插件。
   - 在终端运行 `npx serve .` 或 `python3 -m http.server`。
3. 直接在浏览器打开 `index.html` 亦可运行（由于没有外部模块依赖）。

## GitHub Pages 部署方式
1. 创建一个新的 GitHub 仓库。
2. 将 `index.html` 和 `game.js` 推送到仓库。
3. 进入仓库设置 (Settings) -> Pages。
4. 选择 `main` 分支并点击保存。
5. 等待部署完成后，通过 `https://<your-username>.github.io/<repo-name>/` 访问。
