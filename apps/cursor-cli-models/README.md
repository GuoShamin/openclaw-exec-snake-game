# Cursor CLI · 多模型贪吃蛇自动驾驶对比

同一份 prompt，在**完全隔离**的工作区中分别用不同模型生成的「自动驾驶贪吃蛇（移动端适配 + GitHub Pages 友好）」实现。

- 入口页：`apps/cursor-cli-models/index.html`
- 每个模型一个子目录：`apps/cursor-cli-models/<model>/`
- 对照 prompt：`apps/cursor-cli-models/PROMPT.md`
- prompt SHA256：`apps/cursor-cli-models/PROMPT.sha256`

> 说明：个别模型会输出“文件内容”而不是直接落盘；此仓库中对应目录的文件内容均来自模型原始输出的**机械提取/落盘**，未做人工改写。
