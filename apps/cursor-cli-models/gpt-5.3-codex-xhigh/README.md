# Snake 自动驾驶（静态网页）

一个可直接部署到 GitHub Pages 的贪吃蛇小游戏。  
默认自动开始并自动驾驶，也支持手动接管（键盘/触屏滑动/按钮）。

## 自动驾驶策略（简述）

代码在 `app.js` 里使用了混合策略：

1. 用 BFS 寻找蛇头到食物的最短路；
2. 先模拟“按该路径吃到食物”后的蛇身，再检查蛇头是否仍能到达蛇尾（避免自困）；
3. 若不安全，则用 BFS 追尾巴以延长生存；
4. 若仍无路，选择“可达空间更大且更靠近食物”的安全方向兜底。

## 本地运行（最简）

方式 1（推荐，启动静态服务器）：

```bash
python3 -m http.server 8000
```

然后在浏览器打开 `http://localhost:8000/`。

方式 2：直接打开 `index.html` 也可运行（但推荐方式 1，行为更接近线上环境）。

## GitHub Pages 部署

这是纯静态站点，无需构建步骤，资源路径均为相对路径（如 `./app.js`、`./styles.css`）。

常见部署方式（`docs/`）：

1. 将本目录文件放到仓库的 `docs/` 目录；
2. GitHub 仓库设置中打开 **Pages**；
3. Source 选择 `Deploy from a branch`；
4. Branch 选 `main`（或你的默认分支），Folder 选 `/docs`；
5. 保存后等待部署完成，访问：
   - `https://<你的用户名>.github.io/<仓库名>/`

如果你将文件放在仓库根目录并用根目录发布，也可以直接访问同样的 Pages 地址。
